First Version of Face Tracking Framework

Network Structure

Using object queries to strengthen the face features

Object queries in the upper hand is the learnable parameters input into the transformer to learn the face features.

CFA to obtain the relative position of the face in the search image

The function of CFA is to merge the template and search region, which encode the relative position of template and search region. By the CFA structure, the feature of the face from search region is also obtained.

Decode box to process the position and face for the given search region

Referenced by TransT, there should be one more transformer block before separate the bounding box regression network and predicted the faces.

Training Strategy

CosFace loss

After the regression network, we prefer to use CosFace loss function to strengthen the training of different faces, which empirically has better performance than entropy loss.

Dataset

Based on Youtube face dataset, we randomly pick up the image pair with the same person or from different person with equal probability.
